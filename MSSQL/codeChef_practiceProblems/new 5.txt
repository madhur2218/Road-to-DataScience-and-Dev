Q5: Write a query to find the total count of books whose genre is Fiction.
	Note: Output column name should be fiction_count.
A5: SELECT COUNT(genre) AS fiction_count FROM Books

Q7: Create a query to retrieve the employee_name, company, and salary for employees in the full-time category, ordered by salary in descending order
A7: SELECT employee_name, company, salary FROM Employees WHERE category = 'Full-Time'  ORDER BY salary DESC 

Q8: Write a query to group the employees by their department and display the total number of employees (as total_employees) in each department.
A8: SELECT department, COUNT(employee_name) AS total_employees FROM Employees
	GROUP BY department
	
Q9:Write a query to retrieve the author_id, author_name, and publication_name for authors whose articles got zero views. The result should be sorted by author_id in ascending order.
Return the result table sorted by id in ascending order.
A9: SELECT author_id, author_name, publication_name FROM Views WHERE view_count = 0  ORDER BY author_id ASC
